# Аналитика по Транзакциям
## Разберем модель *Транзакции*
- id - айди транзакции, генерируемый БД
- amount - сумма транзакции
- date - дата **совершения** транзакции (например, транзакция за прошлый понедельник)
- createdAt - дата создания транзакции
- category - @ManyToOne только непонятно зачем, ведь для одной транзакции одна категория
- description - описание

Используется @PrePersist для установки значений createdAt
Метод *getType()* возвращает тип категории, который в дальнейшем позволяет определить тип самой транзакции ()
